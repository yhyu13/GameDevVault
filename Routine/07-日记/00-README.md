## 每日记录流程

### 早晨（5 分钟）

打开 [[周模板]]，查看今天的主题：

| 星期 | 主题 | 对应知识库 |
|------|------|-----------|
| 周一 | 前沿技术与论文复现 | [[01-论文笔记库]] / [[02-引擎源码分析库]] |
| 周二 | 专项技能突破 | [[03-Shader与特效案例集]] + 数学复习 |
| 周三 | 强制休息/游戏游玩 | 日记中记录观察 |
| 周四 | 工程化与工具链 | [[04-性能优化备忘录]] |
| 周五 | 轻量复盘 | 整理归档 + [[05-技术雷达]] |
| 周六 | 项目实战（上午） | 输出文档/博客 |
| 周日 | 项目实战（下午） | 输出文档/博客 + [[99-Templates/周复盘]] |

### 晚间（10-15 分钟）

使用 [[99-Templates/每日日记]] 模板记录：

1. **今日做了什么** — 具体、可衡量
2. **输入了什么** — 读了几页/看了哪个视频/读了哪段源码
3. **输出了什么** — 笔记/代码/Demo/博客
4. **连接了什么** — 和哪些人交流了
5. **明日预告** — 提前准备，降低启动阻力

### 周末（30-60 分钟）

使用 [[99-Templates/周复盘]] 模板：

1. 本周完成度评分
2. 亮点与卡点
3. 知识库更新情况
4. 下周调整计划
