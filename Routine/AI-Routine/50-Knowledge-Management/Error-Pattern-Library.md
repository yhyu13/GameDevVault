# 错误模式库

> **目标**：收集 Agent 常见错误模式，建立快速诊断索引
> **更新频率**：每次发现新问题即记录

---

## 错误分类

### 1. 工具层错误

#### 1.1 文件操作错误
| 错误信息 | 根因 | 解决方案 | 预防 |
|----------|------|----------|------|
| `old_string not found` | 字符串不匹配（大小写、空格、换行符） | 使用 `Read` 确认实际内容，精确匹配 | 编辑前必读文件，从 Read 输出复制 |
| `path not found` | 路径错误、文件已移动/删除 | 检查路径，使用 `Glob` 或 `ls` 确认 | 操作前确认文件存在 |
| `file too large` | 超出 1000 行/100KB 限制 | 使用 `line_offset` 分页，或 `PythonRun` 处理 | 大文件处理前检查大小 |

#### 1.2 命令执行错误
| 错误信息 | 根因 | 解决方案 | 预防 |
|----------|------|----------|------|
| `Command failed with exit code: 1` | 命令参数错误、依赖缺失 | 检查命令语法，确认依赖安装 | 命令执行前验证参数 |
| `timeout` | 任务耗时超过限制 | 拆分任务，增加 `timeout` 参数 | 预估任务耗时，合理设置超时 |
| `permission denied` | 权限不足 | 检查文件权限，避免敏感文件操作 | 了解工作区权限边界 |

#### 1.3 Python 执行错误
| 错误信息 | 根因 | 解决方案 | 预防 |
|----------|------|----------|------|
| `ModuleNotFoundError` | 依赖未安装 | 安装缺失包 | 运行前确认依赖 |
| `SyntaxError` | 代码语法错误 | 检查代码，尤其是 `Edit` 生成的代码 | 复杂代码用 PythonRun 验证 |
| `JSON serialization error` | 返回了不可 JSON 序列化的对象 | 转换或提取可序列化部分 | 明确返回类型 |

### 2. 技能层错误

#### 2.1 触发问题
| 现象 | 根因 | 解决方案 |
|------|------|----------|
| 技能未被触发 | 触发关键词不匹配、用户意图模糊 | 优化触发条件，添加更多同义词 |
| 错误技能被触发 | 触发条件重叠 | 明确技能边界，差异化触发条件 |
| 技能触发延迟 | 系统负载高 | 优化技能描述，使其更精确 |

#### 2.2 执行问题
| 现象 | 根因 | 解决方案 |
|------|------|----------|
| 步骤卡住 | 等待用户输入、工具无响应 | 检查步骤是否有明确的完成条件 |
| 结果不符合预期 | 步骤描述模糊、示例不足 | 添加更多示例、明确输出格式 |
| 死循环 | 步骤间循环依赖 | 检查步骤逻辑，确保有终止条件 |

### 3. 上下文层错误

| 现象 | 根因 | 解决方案 |
|------|------|----------|
| 遗忘用户目标 | 上下文过长，关键信息被压缩 | 主动重申目标，使用状态机表示 |
| 误解用户意图 | 指代不清、歧义 | 主动澄清，使用结构化确认 |
| 重复之前的工作 | 未正确更新状态 | 使用 TODO 列表追踪进度 |

### 4. 知识层错误

| 现象 | 根因 | 解决方案 |
|------|------|----------|
| 引用过期信息 | Vault Memory 未及时更新 | 定期审查 Vault，标记过时信息 |
| 事实错误 | 数据源返回错误，或理解偏差 | 多源验证，不确定时明确说明 |
| 混淆相似概念 | 知识边界模糊 | 建立概念索引，明确定义 |

## 快速诊断索引

| 症状 | 可能类别 | 优先检查 |
|------|----------|----------|
| 工具调用失败 | 工具层 | 参数、路径、权限、超时 |
| 输出格式错误 | 技能层 | 步骤描述、示例、模板 |
| 答非所问 | 上下文层 | 意图理解、目标重申、澄清 |
| 信息不准确 | 知识层 | 数据源、时效性、多源验证 |
| 循环/卡死 | 技能/上下文 | 步骤依赖、终止条件、状态追踪 |
