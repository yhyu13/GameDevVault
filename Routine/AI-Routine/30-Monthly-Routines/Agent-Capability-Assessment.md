# Agent 能力评估

> **执行时间**：每月一次
> **预计耗时**：1-2 小时
> **目的**：系统性评估 Agent 当前能力边界，识别短板，制定下月进化计划

---

## 评估维度

### 1. 工具覆盖度
- [ ] 已掌握工具数 / 可用工具数
- [ ] 工具使用熟练度（能否组合使用？）
- [ ] 工具降级/替代方案储备

### 2. 技能成熟度
- [ ] 已注册技能数
- [ ] 技能质量评级（A/B/C/D）
- [ ] 技能触发准确率（用户意图匹配度）

### 3. 编码能力
- [ ] 支持语言：Python / C++ / C# / HLSL / GLSL / ...
- [ ] 代码质量：可读性、健壮性、可维护性
- [ ] 调试能力：能否自我诊断和修复代码问题？
- [ ] 架构能力：能否设计模块化、可组合的代码？

### 4. 知识管理
- [ ] 记忆准确率
- [ ] 知识检索速度
- [ ] 知识更新及时性

### 5. 交互质量
- [ ] 响应准确性
- [ ] 多轮对话保持能力
- [ ] 上下文理解深度
- [ ] 用户满意度（基于反馈统计）
