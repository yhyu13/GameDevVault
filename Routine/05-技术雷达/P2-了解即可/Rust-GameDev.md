## 一句话简介

> Rust 是一门系统级编程语言,通过所有权和借用检查器在编译期保证内存安全,无需 GC。在游戏引擎中可用于替代 C++ 的高风险模块(如资源系统、网络层),或作为脚本层嵌入。
