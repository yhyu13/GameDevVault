## 一句话简介

> 输入一段文字描述或一张图,实时生成"可交互的虚拟世界" — **Google Genie 3 已经在 2026 年初展示了"实时生成的可探索 3D 世界",Decart Oasis 把 Minecraft 直播流改造成"实时 AI 生成的世界",SITS2026 把它和 AGI 智能体绑定为下一代游戏智能基础设施**。技术上极前沿,商业化至少还要 1-2 年。

## 成熟度评估

| 维度 | 评分 (1-5) | 说明 |
|------|------------|------|
| 生产可用性 | 1 | 演示阶段,无商业产品 |
| 文档完善度 | 2 | 学术论文 + 演示视频,工程文档稀缺 |
| 社区活跃度 | 4 | 学术 + 大厂研究团队极度活跃,2026 年是爆发期 |
| 学习资源 | 3 | 论文 + GTC 演讲可读,工程实现稀缺 |
| 与现有栈兼容性 | 1 | 和传统引擎架构完全分离,需要新的渲染管线 |

**2026 年主流项目:**
- **Google Genie 3**:从单张图 + 文字描述实时生成可探索 3D 世界
- **Decart Oasis**:24fps 实时 AI 生成"Minecraft 风格"游戏
- **腾讯 Hunyuan World**:开源世界模型,从文本/图像生成 3D 场景
- **World Labs**:生成可持久化的 3D 场景(3DGS + LLM 决策)
- **Sora / Veo**(OpenAI / Google):视频生成,虽然不是 3D,但是同源技术

**和 [[3DGS-Gaussian-Splatting]] 的关系:**
- 3DGS 是"从已存在的场景学到表示"
- World Model 是"从无到有实时生成场景"
- 两者最终会融合 — World Model 用 3DGS 当场景表示

**技术栈关键:**
- **Diffusion + Transformer**:视频扩散模型 + 自回归
- **神经辐射场 / 3DGS**:场景表示
- **实时推理优化**:FP8 / NVFP4 / TensorRT
- **延迟预算**:世界模型要求 60+ fps,当前 24-30 fps 是极限

## 我的判断(只对你说)

**这是 5-10 年后的游戏,不是现在的**。  
短期对你 day-job 没杠杆,但**长期决定你会不会被"AI Native Game Studio" 颠覆**。
