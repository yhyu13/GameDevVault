## 如何使用模板

### 方法一：手动复制（推荐新手）

1. 在目标文件夹中新建 Markdown 文件
2. 打开对应的模板文件
3. 全选复制 → 粘贴到新文件
4. 修改 `{{title}}`、`{{date}}` 等占位符

### 方法二：Obsidian 模板插件（推荐熟练后）

1. 安装官方插件 **Templates**（或社区插件 **Templater**）
2. 设置模板文件夹为 `99-Templates`
3. 新建文件时按 `Alt+T`（Templates 插件）自动插入
4. 使用 Templater 可以实现自动填充日期、自动创建日记文件等高级功能

## 占位符说明

| 占位符 | 含义 | 示例 |
|--------|------|------|
| `{{title}}` | 文件标题/主题 | Lumen Paper |
| `{{date}}` | 当前日期 (YYYY-MM-DD) | 2025-01-06 |
| `{{date:YYYY-MM}}` | 年月 | 2025-01 |
| `{{weekday}}` | 星期几 | 周一 |
| `{{monday}}` | 本周一日期 | 2025-01-06 |
| `{{week_start}}` | 本周开始日期 | 2025-01-06 |
| `{{week_end}}` | 本周结束日期 | 2025-01-12 |

> 使用 Templater 插件时，这些占位符会自动替换。手动复制时，自行替换即可。

## 自定义模板

当你发现某个记录反复出现、但现有模板不满足时，可以：

1. 复制最接近的模板
2. 修改后保存到本文件夹
3. 在 [[00-README]] 的模板清单中更新

## 模板设计原则

1. **结构化 > 自由写** — 每个模板都有固定的字段，防止遗漏关键信息
2. **链接友好** — 模板中预留了 `[[关联知识库]]` 的位置，养成链接习惯
3. **输出导向** — 每个模板都包含「输出产物」检查项，倒逼行动
4. **面试意识** — 论文模板和复盘模板专门设计了「面试谈资」区域
