# 💼 面试与职业复盘日志

> **对应周计划：每周外部接触 — 技术沙龙、面试、Code Review**

---

## 记录策略

**每一次技术交流都是学习机会。** 记录：

1. **被问到的盲点** — 这是最宝贵的输入
2. **我当时的回答** — 诚实记录，不美化
3. **更优答案** — 事后查阅资料后的正确答案
4. **行动计划** — 针对性补强
