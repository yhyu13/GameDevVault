## 收集策略

不要收藏夹吃灰。论文进入此库前，必须回答：

1. **这篇论文解决什么问题？**（一句话）
2. **核心创新点是什么？**（与其他方案的区别）
3. **对我当前工作是否有直接帮助？**（P0/P1/P2）
4. **是否值得复现？**（时间成本 vs 收益）
