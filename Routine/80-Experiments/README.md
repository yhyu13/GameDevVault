# 实验沙盒

> **定位**：放置实验性的 Agent 功能、技能、工具原型，不影响生产环境
> **规则**：
> 1. 实验必须标记状态和结论
> 2. 失败实验同样有价值，记录失败原因
> 3. 成功实验可提升到 `40-Agent-Coding-Tasks/` 或正式技能目录
> 4. 定期清理（每月）过期的实验

## 实验模板

创建新实验时，按此模板创建文件：

```markdown
# 实验：[实验名]

## 背景
[为什么要做这个实验]

## 目标
[期望达成的结果]

## 假设
[如果...那么...]

## 方法
[具体步骤]

## 资源
- 文件：...
- 工具：...
- 环境：...

## 结果
[实际观察到的结果]

## 结论
- 成功/失败/部分成功
- 原因分析

## 后续
- [ ] 提升到正式功能
- [ ] 继续改进
- [ ] 废弃归档
```
