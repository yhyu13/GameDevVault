# 🚀 快速上手指南

> 第一次使用此 Vault？按以下步骤 10 分钟完成初始化。

---

## Step 1：打开 Vault

1. 下载并安装 [Obsidian](https://obsidian.md/)
2. 打开 Obsidian → "Open folder as vault" → 选择 `D:\GitRepo-My\GameDevVault`
3. 信任作者 → 启用社区插件（可选）

---

## Step 2：安装推荐插件（可选但强烈推荐）

| 插件 | 用途 | 安装方式 |
|------|------|----------|
| **Templater** | 自动填充日期、变量、动态内容 | 社区插件市场 |
| **Dataview** | 自动汇总查询（如"本月读了几篇论文"） | 社区插件市场 |
| **Calendar** | 日历视图快速跳转到日记 | 社区插件市场 |
| **Git** | 自动备份到 Git（已在 git repo 中） | 社区插件市场 |
| **Graph Analysis** | 知识图谱可视化 | 社区插件市场 |

> 插件安装路径：设置 → 第三方插件 → 关闭安全模式 → 浏览 → 搜索安装 → 启用

---

## Step 3：启用自定义 CSS（已配置）

1. 设置 → 外观 → CSS 代码片段
2. 启用 `custom-colors`
3. 重启 Obsidian

效果：不同标签会有不同颜色（如 `#paper/xxx` 蓝色、`#shader/xxx` 粉色等）。

---

## Step 4：创建你的第一篇笔记

### 路径 A：从日记开始（最简单）

1. 打开 [[07-日记]] → 打开 [[周模板]]
2. 复制内容，创建本周的周计划文件
3. 打开 [[99-Templates/每日日记]]，复制内容创建今天的日记
4. 按照模板填写

### 路径 B：从论文开始（最有价值）

1. 打开 [[01-论文笔记库]]
2. 打开 [[99-Templates/论文笔记]]
3. 复制内容，创建你的第一篇论文笔记
4. 推荐首篇：UE5 的 Lumen、Nanite，或你正在读的其他论文

## 常见问题

**Q：标签怎么用？**  
A：在笔记任意位置写 `#tag/子标签`，如 `#paper/lumen`。Obsidian 会左侧标签面板自动归类。

**Q：双向链接怎么建？**  
A：打字时输入 `[[`，会弹出搜索框，选择要链接的笔记。即使笔记不存在，回车也会自动创建。

**Q：图片放在哪里？**  
A：建议每个知识库自建 `assets/` 子文件夹，或统一放在 `99-Assets/`。

**Q：怎么同步到手机？**  
A：方案一：Git + GitHub（免费）；方案二：Obsidian Sync（官方，付费）；方案三：iCloud/Dropbox 同步文件夹。
